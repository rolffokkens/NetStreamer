Win32 Client
You can download the binary distribution of the Win32 NetStreamer frontend from ftp://flits102-126.flits.rug.nl/pub/NetStreamer/win32. The source distribution will follow later.
It appears that this win32 client doesn't run well on Win95: you can't see any buttons, so you'll need WinNT. This will be addressed later.
